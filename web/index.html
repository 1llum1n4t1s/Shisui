Features

散らかった設定を、静かに整える。Quietly tidy up your network settings.

Everything network, in one calm place.Everything network, in one calm place.

DNS の切り替えからキャッシュのクリア、Windows では BBR2 / TCP の調整やメンテナンスコマンド、ゴーストデバイスの掃除まで。コマンドプロンプトを開かずに、澄んだ水面のようなネットワークへ。From DNS switching and cache flushing to — on Windows — BBR2 / TCP tuning, maintenance commands and ghost-device cleanup. No command prompt required.

DNS Provider Switching

DNS プロバイダの切り替えSwitch DNS providers

Cloudflare（標準 / マルウェアブロック / マルウェア+アダルトブロック）や Google Public DNS、任意のカスタムサーバーへワンクリックで切り替え。IPv4 / IPv6 の両方に対応し、「自動取得（DHCP）に戻す」もボタンひとつ。Switch to Cloudflare (standard / malware-block / malware+adult-block), Google Public DNS, or any custom server in one click. Full IPv4 & IPv6 support, plus a one-tap "back to automatic (DHCP)".

DNS キャッシュのクリアFlush DNS cache

BBR2 Congestion Control

BBR2 輻輳制御の切り替えBBR2 congestion control

TCP の輻輳制御を BBR2 に切り替え。いま有効か無効かがバッジで一目で分かります。RSC / ECN / Timestamps / RSS も個別に切り替え可能。Switch TCP congestion control to BBR2, with a badge showing at a glance whether it's on. RSC / ECN / Timestamps / RSS can each be toggled individually.

ネットワークメンテナンスMaintenance commands

nbtstat / ipconfig / netsh / route / arp などの定番コマンドをカタログから選んで実行。影響の大きいコマンドは「確認済み」にチェックを入れないと実行できない、安全設計です。Run staple commands — nbtstat / ipconfig / netsh / route / arp — from a curated catalog. High-impact commands stay locked until you tick "confirmed", by design.

Ghost Device Cleanup

不要デバイスの掃除Ghost device cleanup
