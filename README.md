# Shisui

Windows / macOS 向けのネットワーク設定デスクトップアプリです。DNS の切り替え、DNS キャッシュのクリア、
(Windows では) BBR2 輻輳制御や各種ネットワークメンテナンスコマンドの実行をひとつのアプリから行えます。

> ⚠️ 本アプリはネットワークアダプタの設定変更・OS のキャッシュクリア・TCP スタック調整など、
> システムに直接影響する操作を行います。管理者権限 (Windows) / 管理者パスワード (macOS) が必要です。

## 機能

- **自動最適化タブ**: 「おまかせ高速化設定」では、何を設定すればいいか分からない場合にボタン 1 つで DNS を Cloudflare に切り替えて
  DoH を有効化し、DNS キャッシュをクリアします (Windows では NetBIOS 名前・ARP/経路キャッシュの追加クリア、
  Winsock 送信自動調整の有効化、
  他の高速化ツールによる変更を含む TCP 設定と TCP ACK 関連レジストリ値の既定状態への復元、
  ループバック Large MTU の有効化、受信ウィンドウ自動調整のリセットもあわせて行います。TCP ACK 関連
  レジストリ値の反映には、実行後の PC 再起動が必要です)。おまかせ内の各設定変更は、対応する個別画面
  (DNS 設定 / BBR2・TCP 調整 / メンテナンス) から単独でも実行できます。Windows では再起動後に
  「すべて計測」を実行すると、Auto-Tuning / BBR2 / RSC / ECN / RSS / TCPタイムスタンプの6項目を順番に
  比較し、環境に合った推奨構成をまとめて適用できます。各候補は固定3回、合計45回・最大約225MBです。
- **使い込んだPC向けネットワーク診断** (Windows 専用): 選択中NICのエラー・破棄パケット、ドライバー日付、
  Windowsが問題を報告しているネットワークデバイス、切断済みNIC、Winsock送信自動調整、古いタスクオフロード
  設定を読み取り専用で確認します。問題に応じて既存のゴーストデバイス削除・Winsockリセットへ案内し、
  必要な場合だけ選択中NICの詳細設定を工場出荷値へ戻せます。
- **DNS プロバイダの切り替え**: ネットワークアダプタを選んで DNS サーバーを設定します。
  - Cloudflare (標準): `1.1.1.1` / `1.0.0.1`
  - Cloudflare (マルウェアブロック): `1.1.1.2` / `1.0.0.2`
  - Cloudflare (マルウェア + アダルトブロック): `1.1.1.3` / `1.0.0.3`
  - Google Public DNS: `8.8.8.8` / `8.8.4.4`
  - Quad9 (マルウェア・フィッシングブロック): `9.9.9.9` / `149.112.112.112`
  - NextDNS (Linked IP): `45.90.28.0` / `45.90.30.0`
  - カスタム: 上記以外の IPv4 / IPv6 アドレスも自由に指定可能 (入力内容は次回起動時も保持されます)
  - いずれも IPv4 / IPv6 の両方に対応。「自動取得 (DHCP) に戻す」でいつでも元に戻せます。
  - 選択中の DNS への疎通テスト (ping 応答速度の簡易表示) も行えます。
- **DNS over HTTPS (DoH) / DNS over TLS (DoT) の切り替え** (Windows 専用): 対応したプリセットではチェック
  ボックス 1 つで有効化・無効化できます。DoH の表示は常に OS の実際の設定状態を反映します。
- **DNS キャッシュクリア**: ワンクリックで OS の DNS キャッシュを破棄します。
- **ネットワーク診断タブ**: このPC、Cloudflare / Google / Quad9 のDNS、Google、GitHubをドロップダウンから
  選ぶか、任意のホスト名 / IP アドレスを直接入力して Ping・トレースルートを実行できます
  (Windows / macOS 両対応)。
- **ネットワークアダプタの詳細表示**: 選択中アダプタの MAC アドレス・リンク速度を確認できます (読み取り専用)。
- **BBR2 輻輳制御の切り替え** (Windows 専用): TCP の輻輳制御アルゴリズムを BBR2 に変更し、
  ループバックの Large MTU を無効化します。既定値への復元も可能です。
- **受信ウィンドウ自動調整 (Auto-Tuning) の設定と計測** (Windows 専用): 手動で5段階のレベルを切り替えられるほか、
  自動最適化タブの一括計測で各レベルを3回ずつ試し、TCPダウンロード速度が最も高いレベルを推奨します。
- **MTU 設定の復元** (Windows 専用): 高速化ツールなどで変更された対象アダプタの IPv4/IPv6 MTU を標準値の 1500 に戻せます。
- **TCP グローバル詳細設定** (Windows 専用): RSC / ECN Capability / TCP Timestamps / RSS / TCP Fast Open を
  個別に有効化・無効化できます。自動最適化タブではBBR2 / RSC / ECN / TCPタイムスタンプを
  ダウンロード負荷時Ping、RSSをダウンロード速度で有効・無効比較します。計測後は各開始前状態へ戻り、
  明確な差が出た項目は実測値、判定できなかった項目はWindows既定値としてAuto-Tuningとまとめて適用できます。
- **ネットワークメンテナンスコマンド** (Windows 専用): NetBIOS キャッシュの再読込、IP アドレスの再取得、
  プロキシ設定のリセット、Winsock / TCP・IP スタックのリセットなど、任意に実行できるコマンド集です。
  まとめて実行する方が意味のある項目 (IP アドレス再取得など) は一括実行ボタンも使えます。
- **切断済みネットワークデバイスの削除** (Windows 専用): 使われなくなった USB アダプタや VPN の
  残留デバイスを一覧表示し、ワンクリックで削除できます。
- **ネットワーク接続名の連番整理** (Windows 専用): PnP 上で現在接続されていないネットワークデバイス登録を
  すべて削除します。セーフモード等でアダプタ一覧が空でも実行でき、通常起動時にアダプタを選択している場合は
  「イーサネット 2」等の名前も連番なしへ戻せます。
- **自動更新**: 起動時 (設定でオフも可) または手動チェックで新しいバージョンを検出し、
  ダウンロードと適用までアプリ内で完結します。

すべてのコマンド実行結果は画面下部の実行ログにそのまま表示されます。

## 動作環境

| 項目 | 要件 |
| --- | --- |
| OS | Windows 10 / 11、macOS |
| 権限 | 管理者権限 (Windows は起動時に自動で昇格を要求します) |

macOS では DNS の切り替え・DNS キャッシュクリア・ネットワーク診断 (Ping / トレースルート) ・アダプタ詳細表示に
対応しています。DoH/DoT・BBR2 / TCP 詳細設定・メンテナンスコマンドは Windows の `netsh` に依存する機能のため、
macOS では該当タブや項目が表示されません。

## インストール

Windows 版は署名済みPerMachine MSIを配布しています。[https://shisui.nephilim.jp/Shisui-win.msi](https://shisui.nephilim.jp/Shisui-win.msi) を
ダウンロードして実行してください。`Program Files`へ安全にインストールされ、以後は自動更新で最新版に追従します。

旧`Shisui-win-Setup.exe`版を利用中の場合は、更新後の初回起動時にMSI版への移行案内が表示されます。
設定とログは保持され、移行完了後に旧`%LocalAppData%\Shisui`本体・Updater・パッケージキャッシュ・旧ショートカットが削除されます。
また、旧MSIによって`C:\Shisui`または`Program Files\ゆろち\Shisui`へインストールされた環境では、
初回起動時の案内に沿って正規の`Program Files\Shisui`へ修復移行し、旧インストール先を回収できます。

## ビルドして実行する

macOS 版、または開発用にソースからビルドする場合は以下の手順です。

```bash
dotnet build Shisui.slnx
dotnet run --project src/Shisui.UI
```

Windows では起動時に UAC の確認ダイアログが表示されます。許可すると管理者権限でアプリが起動します。

## 使い方

1. **DNS 設定タブ**: 対象のネットワークアダプタと DNS プロバイダを選び、「適用」を押します。
   DoH / DoT に対応したプリセットではチェックボックスで有効・無効も切り替えられます。
2. **自動最適化タブ**: 対象アダプタを選んでクイック最適化を実行します。Windowsでは再起動後に
   「すべて計測」を実行し、結果を確認して「推奨設定を一括適用」を押します。
3. **ネットワーク診断タブ**: よく使う対象を選ぶか、ホスト名や IP アドレスを直接入力して Ping / トレースルートを実行します。
4. **BBR2 / TCP 調整タブ** (Windows): 「BBR2 を有効化」、受信ウィンドウ自動調整、MTU の標準値への復元、個別の TCP 設定を切り替えます。
5. **メンテナンスタブ** (Windows): 必要なコマンドの「実行」を押します。まとめて実行できる
   カテゴリでは「まとめて実行」ボタンも使えます。

## トラブルシュート

- **DNS 設定が反映されない**: 対象アダプタが有効 (接続中) になっているか確認してください。反映後に
  「一覧を更新」で現在の DNS 設定を再取得できます。
- **コマンドが失敗する**: 実行ログに表示されるエラーメッセージを確認してください。管理者権限で起動して
  いない場合、ほぼ全てのコマンドが権限エラーで失敗します。
- **Winsock リセット等を実行した後にネットワークがおかしい**: 「要再起動」と表示されているコマンドは
  PC を再起動するまで完全には反映されません。
