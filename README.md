# Shisui

Windows / macOS 向けのネットワーク設定デスクトップアプリです。DNS の切り替え、DNS キャッシュのクリア、
(Windows では) BBR2 輻輳制御や各種ネットワークメンテナンスコマンドの実行をひとつのアプリから行えます。

> ⚠️ 本アプリはネットワークアダプタの設定変更・OS のキャッシュクリア・TCP スタック調整など、
> システムに直接影響する操作を行います。管理者権限 (Windows) / 管理者パスワード (macOS) が必要です。

## 機能

- **DNS プロバイダの切り替え**: ネットワークアダプタを選んで DNS サーバーを設定します。
  - Cloudflare (標準): `1.1.1.1` / `1.0.0.1`
  - Cloudflare (マルウェアブロック): `1.1.1.2` / `1.0.0.2`
  - Cloudflare (マルウェア + アダルトブロック): `1.1.1.3` / `1.0.0.3`
  - Google Public DNS: `8.8.8.8` / `8.8.4.4`
  - カスタム: 上記以外の IPv4 / IPv6 アドレスも自由に指定可能
  - いずれも IPv4 / IPv6 の両方に対応。「自動取得 (DHCP) に戻す」でいつでも元に戻せます。
- **DNS キャッシュクリア**: ワンクリックで OS の DNS キャッシュを破棄します。
- **BBR2 輻輳制御の切り替え** (Windows 専用): TCP の輻輳制御アルゴリズムを BBR2 に変更し、
  ループバックの Large MTU を無効化します。既定値への復元も可能です。
- **TCP グローバル詳細設定** (Windows 専用): RSC / ECN Capability / TCP Timestamps / RSS / TCP Fast Open を
  個別に有効化・無効化できます。
- **ネットワークメンテナンスコマンド** (Windows 専用): NetBIOS キャッシュの再読込、IP アドレスの再取得、
  プロキシ設定のリセット、Winsock / TCP・IP スタックのリセットなど、任意に実行できるコマンド集です。
  影響の大きいコマンドは「確認済み」チェックを入れないと実行できません。

すべてのコマンド実行結果は画面下部の実行ログにそのまま表示されます。

## 動作環境

| 項目 | 要件 |
| --- | --- |
| OS | Windows 10 / 11、macOS |
| 権限 | 管理者権限 (Windows は起動時に自動で昇格を要求します) |

macOS では DNS の切り替え・DNS キャッシュクリアのみ対応しています。BBR2 / TCP 詳細設定 / メンテナンスコマンドは
Windows の `netsh` に依存する機能のため、macOS では該当タブ自体が表示されません。

## ビルドして実行する

現時点ではインストーラの配布は行っていません。ソースからビルドして実行してください。

```bash
dotnet build Shisui.slnx
dotnet run --project src/Shisui.UI
```

Windows では起動時に UAC の確認ダイアログが表示されます。許可すると管理者権限でアプリが起動します。

## 使い方

1. **DNS 設定タブ**: 対象のネットワークアダプタと DNS プロバイダを選び、「適用」を押します。
2. **BBR2 / TCP 調整タブ** (Windows): 「BBR2 を有効化」または個別の TCP 設定を切り替えます。
3. **メンテナンスタブ** (Windows): 必要なコマンドの「実行」を押します。影響の大きいコマンドは
   先に「確認済み」にチェックを入れてください。

## トラブルシュート

- **DNS 設定が反映されない**: 対象アダプタが有効 (接続中) になっているか確認してください。反映後に
  「一覧を更新」で現在の DNS 設定を再取得できます。
- **コマンドが失敗する**: 実行ログに表示されるエラーメッセージを確認してください。管理者権限で起動して
  いない場合、ほぼ全てのコマンドが権限エラーで失敗します。
- **Winsock リセット等を実行した後にネットワークがおかしい**: 「要再起動」と表示されているコマンドは
  PC を再起動するまで完全には反映されません。
