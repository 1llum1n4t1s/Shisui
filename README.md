# Shisui

Shisuiは、DNSの切り替え、通信状態の診断、Windowsのネットワーク設定の調整を、ひとつの画面から行えるデスクトップアプリです。

設定に詳しくない場合は「おまかせ高速化設定」、自分で確認しながら使いたい場合は各機能を個別に実行できます。

> **ご注意**
> Shisuiは、ネットワークアダプターやWindowsの通信設定を変更します。実行中に一時的に通信が切れることがあります。
> Windowsでは起動時に管理者権限を要求します。

## ダウンロード

### Windows

[Shisuiの最新版をダウンロード](https://shisui.nephilim.jp/Shisui-win.msi)

ダウンロードしたMSIを実行してインストールしてください。以後の更新はアプリ内から適用できます。

旧インストーラー版を利用している場合は、起動時の案内に従って最新版へ移行できます。設定とログは引き継がれます。

### macOS

現在、一般利用者向けのインストーラーは配布していません。

## 主な機能

### Windows / macOS

- **DNSの切り替え**
  Cloudflare、Google Public DNS、Quad9、NextDNS、任意のIPv4／IPv6アドレスを選択できます。いつでも自動取得（DHCP）へ戻せます。
- **DNSキャッシュのクリア**
  古い名前解決情報を削除します。
- **Ping・トレースルート**
  よく使うサービス、任意のホスト名、IPアドレスへの通信状態を確認できます。
- **アダプター情報の確認**
  MACアドレスやリンク速度を表示します。

### Windows

- **おまかせ高速化設定**
  DNSやWindowsの通信設定をShisuiが推奨する状態へまとめて整え、切断済みのネットワークデバイス登録もすべて削除します。実行後はPCを再起動してください。
- **環境に合わせた自動計測**
  Auto-Tuning、BBR2、RSC、ECN、RSS、TCPタイムスタンプを比較し、結果に合った設定を提案します。
- **暗号化DNS**
  対応するDNSでDoH（DNS over HTTPS）とDoT（DNS over TLS）を切り替えられます。
- **BBR2・TCP設定**
  BBR2、受信ウィンドウ自動調整、RSC、ECN、RSS、TCP Fast Openなどを個別に設定できます。
- **MTUの復元**
  変更されているIPv4／IPv6のMTUを標準値の1500へ戻せます。
- **ネットワークメンテナンス**
  IPアドレスの再取得、各種キャッシュのクリア、プロキシ・Winsock・TCP/IP設定のリセットなどを実行できます。
- **使い込んだPC向け診断**
  パケットエラー、古いドライバー、問題のあるネットワークデバイス、過去の高速化設定などを読み取り専用で確認します。
- **接続名の連番整理**
  Windowsが切断済みと判定したネットワークデバイス登録をまとめて削除し、「イーサネット 2」などの連番を外せます。セーフモードなどでアダプター一覧が空でも削除できます。
- **自動更新**
  起動時または手動で最新版を確認し、アプリ内から更新できます。

## おすすめの使い方

### 設定をおまかせしたい場合

1. 「自動最適化」を開き、使用中のネットワークアダプターを選びます。
2. 「おまかせ高速化設定」を実行します。
3. PCを再起動します。
4. より細かく調整したい場合は「すべて計測」を実行します。
5. 結果を確認して「推奨設定を一括適用」を押します。

### DNSだけ変更したい場合

1. 「DNS設定」を開きます。
2. 使用中のネットワークアダプターとDNSを選びます。
3. 「適用」を押します。

元に戻す場合は「自動取得（DHCP）に戻す」を使用してください。

### 通信状態を調べたい場合

「ネットワーク診断」で対象を選び、Pingまたはトレースルートを実行してください。

## 使用前に確認してください

- 「すべて計測」は合計で最大約225MBをダウンロードします。従量制回線では実行しないでください。
- 「おまかせ高速化設定」には、PCの再起動が必要な変更が含まれます。
- WinsockやTCP/IPのリセット、IPアドレスの再取得では、一時的に通信が切れることがあります。
- 「おまかせ高速化設定」と接続名の連番整理では、取り外し中のUSB LANアダプターやドック内蔵LANも削除対象になる場合があります。再接続時にWindowsが再検出します。
- 無効化されているだけの現役デバイスは、接続名の連番整理では削除しません。
- NextDNSを利用する場合は、NextDNS側で現在のIPアドレスをLinked IPとして設定してください。
- コマンドの成否やエラー内容は、画面下部の実行ログで確認できます。

## 動作環境

| 項目 | 内容 |
| --- | --- |
| Windows | Windows 10 / 11（64ビット） |
| macOS | DNS設定、DNSキャッシュクリア、Ping、トレースルート、アダプター情報に対応 |

| インターネット接続 | インストール、更新、自動計測に必要 |

## 困ったとき

- **DNS設定が反映されない**
  対象のネットワークアダプターが有効になっているか確認し、「一覧を更新」を実行してください。
- **コマンドが失敗する**
  画面下部の実行ログを確認してください。Windowsでは管理者権限がないと設定を変更できません。
- **リセット後に通信できない**
  「要再起動」と表示された操作を実行した場合は、PCを再起動してください。
- **元の設定へ戻したい**
  DNSは「自動取得（DHCP）に戻す」、TCP設定は各項目の「既定値に戻す」を使用してください。
